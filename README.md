# heart-diagnostico
Modelo ML para predicción de riesgo cardíaco con explicabilidad

Todo el código fuente del proyecto se encuentra en la carpeta **Sumativa1/**.
- Notebooks y scripts de entrenamiento
- Generación de gráficos (EDA, matrices de confusión, curvas de pérdida)
- Código para modelos (Regresión Logística, Random Forest, SVM, MLP)
